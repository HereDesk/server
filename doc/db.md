# 数据库说明

## 基础权限表

```
├── api                                       // api名称表
├── api_permissions                           // api权限表
├── pages                                     // web页面
├── pages_permissions                         // web页面权限
├── group                                     // 用户角色表
```

## 缺陷相关

```
├── bug_priority                              // 缺陷优先级
├── bug_severity                              // 缺陷严重程度
├── bug_source                                // 缺陷来源
├── bug_status                                // 缺陷状态
├── bug_type                                  // 缺陷类型
```
